# 画質で逆転！？じゃんけんアプリ  
あなたのイラスト画像でじゃんけん勝負！  
画質が良ければ、じゃんけんの勝敗がひっくり返る…！？
⭐️⭐️⭐️v1のアップデートでローカルストレージ機能（.foreach構文も活用）を追加！！！

---

## 概要

**画質で逆転！？じゃんけんアプリ**は、  
ユーザーがアップロードしたイラスト画像を「グー」「チョキ」「パー」のいずれかとして扱い、  
PC（コンピュータ）のランダムなグーチョキパー画像と勝負するブラウザアプリです。

通常のじゃんけんに加え、  
**画像の画質スコア（解像度・縦横比）による“逆転ルール”**が適用されるのが最大の特徴です。

- じゃんけんで負けても、画質が圧倒的に良ければ勝ちに！  
- じゃんけんで勝っても、画質が低いと負けに！  
- 画像スコアが近いと引き分けにも！  

新感覚の「ゲーム × 画像評価」アプリです。



---

## 動作の流れ

1. ユーザーは **自分のイラスト画像を1枚だけアップロード**
2. 「グー」「チョキ」「パー」をボタンで選び、画像に対応付ける
3. PCがランダムにグーチョキパー画像を選択
4. まずは **普通のじゃんけんで勝敗判定**
5. つづいて **画質スコアを比較**  
6. 画質差によって勝敗が逆転・相殺されることがある！

---

## 逆転ルール（画質で勝敗が変わる）

ユーザー vs PC の画質スコアの差に基づき、最終結果を変動させます。

| 元の結果 | スコア差（ユーザー − PC） | 最終結果 | 説明 |
|---------|----------------------------|----------|------|
| 負け | +20 以上 | 勝ち | **大逆転勝利！** |
| 負け | +10 以上 | 引き分け | 画質で粘り切った |
| 勝ち | −20 以下 | 負け | PCの圧倒的画質に敗北 |
| 勝ち | −10 以下 | 引き分け | 画質の差で勝ちを逃す |
| あいこ | +10 以上 | 勝ち | 画質の勝利 |
| あいこ | −10 以下 | 負け | 画質の敗北 |

---

## 画質スコアの仕組み（0〜100点）

以下の要素で加点・減点します。（`script.js` の `evaluateImage()` 内で調整可能）

### 解像度（短辺の長さ）
- **1000px 以上** → +20点  
- **700px 以上** → +10点  
- **400px 以上** → ±0点  
- **400px 未満** → −15点  

### 縦横比（長辺 ÷ 短辺）
- **1.2 未満** → +10点（アイコン向き）  
- **1.8 未満** → +5点（扱いやすい）  
- **2.5 未満** → ±0点  
- **2.5 以上** → −5点  

※ 最終スコアは 0〜100 にクリップされます。
